토큰 발급 방법은 아래 안내를 참고하세요

깃허브 Pages가 활성화된 리포지토리 이름

1. github.com/settings/tokens/new 접속
2. Note: 아무 이름 입력 (예: journey-deep)
3. Expiration: No expiration 선택
4. Select scopes에서 repo 체크박스 선택
5. Generate token 클릭
6. 생성된 ghp_xxx 토큰 복사해서 위에 붙여넣기
⚠️ 토큰은 한 번만 보여요. 꼭 복사해두세요!
